Sistema desarrollado por Jonathan Mangarret .

Requisitos.
XAMPP for Windows 5.6.32
PHP Version 5.6.32
MYSQL Version 5.0
Apache 2.4
Conexion a Internet para carga de librerias bootstrap y jquery.

Instalacion.
Instalar  XAMPP for Windows 5.6.32 (32 Bits)
Descomprimir aite.rar y copiar carpeta aite en c:/xampp/htdocs
En caso de usar GIT, clonar o descargar repositorio https://github.com/jmangarret/aite.git en la carpeta c:/xampp/htdocs
Ejecutar en el navegador la URL http://localhost/phpmyadmin/
Crear una base de datos de nombre aite e importar el archivo aite.sql de la carpeta db.
Ejecutar en el navegador la URL http://localhost/aite/
Ingresar usuario admin:admin y contraseña: admin

Config. de acceso MySQL:
Usuario: root
Pass: (Vacio/Null)

Acceso al sistema:
Usuario: admin
Pass: admin
